Designing a Study with GLIMMPSE

To enable you to quickly perform power or sample size analysis, GLIMMPSE will guide you step by step through the process of specifying your study design.

  • Navigation
  • Inputs
  • help_outline save_alt

The arrow buttons allow you to navigate through the steps in the process. A green arrow indicates that the current inputs on the page are and you may move on.

navigate_before navigate_next navigate_next

If you try to move on when the current inputs on the page are invalid, an error message will tell you what is invalid or missing.

Some steps, for example choosing a Type I error Rate, are required for every study design. Others, for example specifying Repeated Measures, are optional. Depending on the design you specify, the steps given may change.

When adding repeated measures, clustering or predictors to your design, the navigation arrows will be hidden. Follow the directions on the page and navigate using the to return to the main flow.

If you navigate to an earlier step and change your design, any later input which is no longer valid with that change will be reset to default values.

add_box
This button indicates that you can add several values of this type, e.g. you can add up to 10 values for Type I error and a power or sample size will be calculated for each value. You must click the button OR press enter to add a value. it will then be added to the table of those values. Navigating to another step will not add a value in the input field.
edit
Click this button to edit the value it is attached to.
delete_forever
Click this button to delete the value it is attached to.
